		userID::Gender::Age:Occupation::Zip-code

分析系统任务：
	受众分析：
	 1.看过"Lord of the Rings, The(1978)" 用户年龄和性别分布
	 2.年龄在“18-24”的男性年轻人，最喜欢看那10部电影
	 3.得分最高的10部电影，看过电影最多的前10个人；
	 	女性看最多的10部电影，男性看过最多的10电影
